# Profitability-Analysis-KPI-Dashboard-Power-BI-
This project is an independent data case study for a boutique hospitality unit (Bikta BeZayit) focused on improving business profitability through data-driven pricing optimization and guest-retention incentives.
The project includes data modeling, cost allocation, KPI design, profitability analysis, and post-implementation performance comparison (Before vs After).
